# daniyalzia.co.uk

Personal portfolio and project hub.

## Overview

A single-page site serving as the central landing point for my projects, infrastructure, and contact info. Built with Next.js and Tailwind CSS.

## Stack

- **Next.js 16** — App Router
- **Tailwind CSS v4**
- **Framer Motion** — scroll-driven animations

## Sections

- **Hero** — intro and key metrics
- **Projects** — featured work with live links
- **Infrastructure** — homelab overview and service status
- **About** — background, disciplines, timeline
- **Contact** — links and reach

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
